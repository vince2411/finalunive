# Universitarios FC v2.0

Sistema SaaS completo para la gestión de academias de fútbol.

## Funcionalidades
- 📊 Dashboard administrativo con métricas en tiempo real
- 👥 Gestión de jugadores y categorías
- 💰 Control financiero (inscripciones, deudas, facturas)
- 💬 Chatbot con IA (Groq/LLaMA) + escalamiento a asesor humano
- 🔄 Omnicanal (Web, WhatsApp, Instagram, Facebook)
- 📈 Reportes dinámicos con gráficas
- 💱 Calculadora BCV USD/VES en tiempo real

## Stack
- HTML/CSS/JS (Vanilla — sin frameworks)
- Firebase (Firestore + Auth)
- Groq AI (LLaMA 3)
- Chart.js

## Despliegue
Sitio estático — compatible con Vercel, Firebase Hosting, o cualquier CDN.
